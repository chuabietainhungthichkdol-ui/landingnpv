
# Website Giới Thiệu PV Power Cà Mau

Dự án website giới thiệu Công ty Điện lực Dầu khí Cà Mau, dựa trên tài liệu doanh nghiệp chính thức.

## Đặc điểm kỹ thuật
- **Static Site**: Không sử dụng React/Next/Vite.
- **Single File**: Toàn bộ CSS, JS và HTML nằm trong `index.html`.
- **Zero Build**: Không cần bước build phức tạp, chạy trực tiếp trên trình duyệt.

## Hướng dẫn Deploy lên Vercel

1. **GitHub**: Push toàn bộ nội dung repo này lên GitHub.
2. **Vercel**:
   - Truy cập Vercel Dashboard, chọn **Add New Project**.
   - Import Repository từ GitHub.
   - Tại phần **Framework Preset**, chọn **Other**.
   - Bỏ qua các cấu hình Build (Build Command để trống).
   - Nhấn **Deploy**.

Website sẽ được kích hoạt tại URL của Vercel với hiệu suất tối ưu và 0% tỷ lệ lỗi "trắng trang".
